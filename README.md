📌 Description

This is a simple project where I created a custom cursor using HTML, CSS, and JavaScript.  
Instead of the normal mouse pointer, a small circular cursor follows the mouse movement on the screen.  

JavaScript is used to track the mouse position, and CSS is used to style the cursor and create a smooth animation effect.
When the cursor moves over the text, it creates a nice visual blend effect.  

This project helped me understand:   
• JavaScript DOM manipulation  
• mousemove events  
• CSS positioning and animation
